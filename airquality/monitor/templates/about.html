Project Overview

Our IoT-based Weather and Air Quality Monitoring System provides real-time environmental insights. It uses an ESP32 microcontroller combined with DHT11 and MQ135 sensors to track temperature, humidity, and air quality. The system is designed to help users stay informed about pollution levels. Data is displayed on a visually appealing dashboard created using HTML, CSS, and JavaScript. This project is ideal for personal health monitoring, education, and smart city applications.

Components Used

ESP32

A powerful Wi-Fi-enabled microcontroller that forms the heart of the project.

DHT11

A temperature and humidity sensor used to monitor basic environmental data.

MQ135

This air quality sensor detects and monitors the level of harmful gases in the environment.
